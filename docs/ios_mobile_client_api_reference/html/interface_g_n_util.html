+ (NSString*) readUTF8: (NSString *)  filename

Reads a UTF-8 encoded text file into a string.

Reads a UTF-8 encoded file into a string one page (4096 bytes) at a time. When the entire file is read, returns a string that contains the file contents. This method is useful for reading plain text files and XML files.

Parameters
filenameUTF-8 encoded file to read
Returns
String containing the file data. When the file does not exist, or there was a problem reading the file, returns null.
