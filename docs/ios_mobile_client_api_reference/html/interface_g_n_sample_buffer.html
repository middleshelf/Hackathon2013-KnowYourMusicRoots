Detailed Description

Base class for a PCM-encoded audio sample.

Method Documentation

numChannels: (int)  inNumChannels
sampleRate: (int)  inSampleRate 

Creates a buffer for a PCM-encoded audio sample that can be provided to fingerprint generation or recognition operations.

Parameters
inSamplesBuffer containing the PCM audio samples.
inBytesPerSampleThe number of bytes allocated for each sample in the inSamples buffer. For example, 1 for 8-bit mono, 2 for 16-bit mono, and 4 for 16-bit stereo.
inNumChannelsThe number of channels encoded in the audio in the inSamples buffer. For example, 1 for mono and 2 for stereo.
inSampleRateThe sample rate of the audio in the inSamples buffer. Mobile Client supports the following sampling rates: 8000, 11025, 16000, 22050, 24000, 32000, 44100, and 48000.
